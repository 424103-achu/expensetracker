# Personal Expense Tracker

Full-stack personal expense tracker with shared expense settlement.

## Stack
- Frontend: React + Vite + Tailwind + Chart.js
- Backend: Node.js + Express + PostgreSQL
- Auth: JWT

## Run
1. Create DB and run `database/schema.sql`.
2. Configure `backend/.env` from `backend/.env.example`.
3. Install dependencies:
   - `npm install`
   - `npm --prefix backend install`
   - `npm --prefix frontend install`
4. Start both apps:
   - `npm run dev`
